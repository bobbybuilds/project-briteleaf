# Pinterest Pipeline

## Purpose

Drive traffic from Pinterest to blog posts using SEO-based pins.

## Steps

1. Select blog post  
2. Create 4–5 pin variations  
3. Write SEO titles and descriptions  
4. Add alt text and tags  
5. Publish to boards  
6. Link directly to blog post  
7. Monitor performance  

## Output

- 4–5 pins per post  
- Multiple boards  
- Direct traffic to site  

## Rule

Every blog post goes through this pipeline before moving on.
